## SEO Guidelines

### On-Page
- Title: Primary keyword + benefit (60 chars max)
- Meta description: Action-oriented (155 chars)
- H1: Include primary keyword
- H2-H4: Include secondary keywords
- Images: Alt text with keywords
- Internal links: 3-5 per post

### Technical
- Schema markup: Article, HowTo, FAQ
- Canonical URLs
- Mobile-first design
- Core Web Vitals: LCP <2.5s, FID <100ms, CLS <0.1
